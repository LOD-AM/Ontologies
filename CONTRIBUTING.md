# Contributing to LOD-AM/ontologies

We welcome contributions! See README.md for details on repository structure.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Standards

- Use kebab-case for files and directories
- Include YAML frontmatter in documentation
- Use Turtle for RDF examples
- Use SPARQL for query examples
