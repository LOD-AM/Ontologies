# Related Projects

## Linked Open Data in Archaeology and Museums (LOD-am)
- **Website**: https://lod-am.net/
- **Description**: Main LOD-AM project website
- **Focus**: Linked Open Data for Ancient Metallurgy
- **Relation**: Parent project of this repository

## ARIADNE
- **Website**: https://www.ariadne-infrastructure.eu/
- **Description**: Advanced Research Infrastructure for Archaeological Dataset Networking in Europe
- **Focus**: Archaeological data integration and access
- **Relation**: Inspires our approach to data integration
- **Ontologies**: Uses CIDOC CRM and extensions

## CLAROS
- **Website**: https://www.clarosnet.org/
- **Description**: World of Ancient Art on the Semantic Web
- **Focus**: Ancient art and archaeology
- **Relation**: Similar domain, uses CIDOC CRM
- **Status**: Archive (project completed)

## CARARE
- **Website**: https://carare.eu/
- **Description**: Connecting Archaeology and Architecture in Europe
- **Focus**: Archaeological monuments and sites
- **Relation**: Uses CIDOC CRM for cultural heritage

## Europeana
- **Website**: https://www.europeana.eu/
- **Description**: European cultural heritage platform
- **Focus**: Digital cultural heritage from European institutions
- **Relation**: Uses EDM (Europeana Data Model) which maps to CIDOC CRM

## Getty Vocabularies
- **Website**: https://www.getty.edu/research/tools/vocabularies/
- **Description**: Controlled vocabularies for cultural heritage
- **Vocabularies**:
  - AAT (Art & Architecture Thesaurus)
  - TGN (Getty Thesaurus of Geographic Names)
  - ULAN (Union List of Artist Names)
  - CONA (Cultural Objects Name Authority)
- **Relation**: Potential vocabulary source for types, materials, etc.

## CIDOC CRM Special Interest Groups
- **Website**: https://cidoc-crm.org/special-interest-groups
- **Description**: Community groups working on CIDOC CRM extensions
- **Relevant SIGs**:
  - CRM-SIG: Main CIDOC CRM group
  - FRBRoo SIG: FRBRoo development
  - CRM-EH: Cultural Heritage
  - CRM-Geo: Geographic information
- **Relation**: Community for ontology development and support

## Allotrope Foundation Community
- **Website**: https://www.allotrope.org/community
- **Description**: Community for Allotrope Framework users and developers
- **Relation**: Support for AFO implementation

## W3C Semantic Web
- **Website**: https://www.w3.org/standards/semanticweb/
- **Description**: W3C Semantic Web standards and resources
- **Relevant Standards**:
  - RDF: Resource Description Framework
  - OWL: Web Ontology Language
  - SPARQL: Query Language for RDF
  - SHACL: Shapes Constraint Language
  - SKOS: Simple Knowledge Organization System
- **Relation**: Foundation standards for our work
